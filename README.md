# Moein Fayegh — Portfolio (Next.js)

سایت شخصی معین فایق، ساخته‌شده با **Next.js 14 (App Router) + TypeScript + Tailwind CSS**.
دوزبانه (فارسی/انگلیسی)، ری‌اکت‌محور، با تم "محیط کد/ترمینال".

## اجرا روی سیستم خودتون

```bash
npm install
npm run dev
```

بعد سایت رو روی `http://localhost:3000` ببینید.

برای بیلد نهایی:

```bash
npm run build
npm run start
```

## فرم تماس — چطور کار می‌کنه؟

فرم بخش تماس به یک API route داخلی در مسیر `app/api/contact/route.ts` وصله.
وقتی کسی فرم رو پر و ارسال می‌کنه:

1. اطلاعات (`name`, `phone`, `email`, `message`) به `POST /api/contact` فرستاده می‌شه.
2. سرور اعتبارسنجی می‌کنه (نام، شماره تماس و پیام الزامی هستن).
3. پیام به همراه تاریخ دریافت و یک شناسه یکتا، داخل فایل `data/messages.json` ذخیره می‌شه.

برای دیدن پیام‌های ذخیره‌شده، وقتی سایت روی سیستم خودتون (یا یک سرور واقعی) در حال اجراست، کافیه به آدرس زیر برید:

```
http://localhost:3000/api/contact
```

### ⚠️ نکته مهم درباره هاست‌های Serverless (مثل Vercel)

اگه سایت رو روی Vercel یا هر هاست serverless دیگه‌ای دیپلوی کنید، فایل‌سیستم فقط-خواندنیه و بین درخواست‌ها هم پایدار نمی‌مونه — یعنی ذخیره‌سازی روی فایل کار نمی‌کنه (یا حداکثر موقتیه). برای اون حالت دو گزینه دارید:

- **ایمیل:** یک بخش نمونه (کامنت‌شده) داخل `app/api/contact/route.ts` گذاشتم که با یک سرویس مثل [Resend](https://resend.com) یا Postmark پیام رو مستقیم به ایمیل شما می‌فرسته. کافیه اکانت بسازید، `RESEND_API_KEY` رو در `.env.local` بذارید و اون بخش رو از حالت کامنت خارج کنید.
- **دیتابیس:** اگه سایت رو روی یک سرور شخصی (VPS) با Node.js اجرا کنید، همین روش فایل کاملاً کار می‌کنه و نیازی به تغییر نیست.

## ساختار پروژه

```
app/
  layout.tsx        ← فونت‌ها، متادیتا، Providers
  page.tsx           ← چیدمان صفحه اصلی
  globals.css
  api/contact/route.ts  ← دریافت و ذخیره پیام‌های فرم
components/
  Providers.tsx       ← Context زبان فارسی/انگلیسی
  EditorBar.tsx        ← نوار بالا (تب‌های فایل + سوییچ زبان)
  Hero.tsx, About.tsx, Skills.tsx, Projects.tsx, Contact.tsx, ContactForm.tsx
  Footer.tsx, TopButton.tsx, RevealOnScroll.tsx
data/
  messages.json        ← پیام‌های ذخیره‌شده (خودکار ساخته و به‌روزرسانی می‌شه)
```

## تغییر زبان و متن‌ها

هر بخش، متن فارسی/انگلیسی رو مستقیم به شکل `t("متن فارسی", "English text")` نگه می‌داره — کافیه همون رشته‌ها رو داخل هر کامپوننت ویرایش کنید.

## لینک‌ها و اطلاعات تماس واقعی

اطلاعات تماس (اینستاگرام، واتساپ، تلگرام، شماره تماس) و لینک نمونه‌کارها داخل `components/Contact.tsx` و `components/Projects.tsx` قرار دارن — لینک StylePikaso فعلاً placeholder (`#`) هست، وقتی آدرس واقعیش رو داشتید جایگزین کنید.
